================================================================================
README - Manic Pietro v1.0
================================================================================
Este juego fue compilado usando el Pietro Framework, programado en C utilizando
z88dk SDCC y la librería NIRVANA+.
================================================================================

HISTORIA

Ayuda a Pietro en su aventura, por 20 diferentes nuevos niveles, basados
en algunos de nuestros favoritos juegos en 8 bits de los 80's.

Los valientes que logren finalizar esta aventura, obtendrán un código
secreto que les dará el acceso a desbloquear los 20 niveles originales de Manic
Miner.

Si mueres en tu misión, se te dará un código para continuar jugando desde la
última etapa, con las vidas que tenías al iniciarla. Pero recuerda no hay
pez espada para los tramposos.

Un remake de la obra maestra "Manic Miner" de el genio Matthew Smith,
para el concurso ZX-DEV M.I.A-Remakes.

CARACTERÍSTICAS
================================================================================
- Solo para ZX Spectrum 128kb (128, +2, +2a, +3).
- Versión personalizada de NIRVANA+ con transparencias de PAPER.
- Versión compatible con modelos Pentagon.
- 40 niveles con distintos tileset.
  -20 niveles únicos.
  -20 niveles clásicos.
- Lenguage Español / Inglés.
- 2 Introducciones.
- 2 Finales.
- 3 AY Songs.
- 13 distintos FXs AY.
- 59 distintos enemigos.
- Códigos para continuar de tu progreso.
- Codigos secretos para desbloquear algunos características ocultas.
- Control Sinclair / Teclado / Kempston.
- Teclado Redefinible.
- Teclas Extra
  - Caps Shift : Cambiar Musica / Musica OFF.
  - Symbol Shift : Pausa.
